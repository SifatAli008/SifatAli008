# Applying AI-WIP proposals

All AI-WIP Deploys commit to the shared branch **`aiwip/wip-review`**
(single PR: “AI-WIP: SEO review (WIP)”).
Proposal files live under `docs/ai-wip-proposals/`.
Review each file, then apply the suggested SEO/content change in the Next.js app
(see `docs/AI_WIP.md` and PLATFORM.md).

AI-WIP never pushes to `main` automatically — you merge the WIP review PR.
